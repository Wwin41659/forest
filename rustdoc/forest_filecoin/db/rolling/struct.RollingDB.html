    db_root: PathBuf,
    db_config: ParityDbConfig,
    db_index: RwLock<FileBacked<DbIndex>>,
    current: RwLock<Arc<ParityDb>>,
    old: RwLock<Arc<ParityDb>>,
}
Expand description

This DB wrapper is specially designed for supporting the concurrent, semi-space GC algorithm that is implemented in DbGarbageCollector, containing a reference to the old DB space and a reference to the current DB space. Both underlying key-vale DB are supposed to contain only block data as value and its content-addressed CID as key
