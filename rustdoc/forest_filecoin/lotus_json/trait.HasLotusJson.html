fn snapshots() -> Vec<(Value, Self)>

To ensure code quality, conversion to/from lotus JSON MUST be tested. Provide snapshots of the JSON, and the domain type it should serialize to.

Serialization and de-serialization of the domain type should match the snapshot.

If using decl_and_test, this test is automatically run for you, but if the test is out-of-module, you must call assert_all_snapshots manually.
