Expand description

Wrap a bare function with our argument parsing logic. Turns any async fn foo(ctx, arg0...) into a function that can be passed to jsonrpsee::server::RpcModule::register_async_method.

This trait is NOT designed to implemented on anything except bare handler functions, as below.
