Charge for the initial memory and tables before a Wasm module is instantiated.

The Wasm instrumentation machinery via [fvm_wasm_instrument::gas_metering::MemoryGrowCost] only charges for growing the memory beyond the initial amount. It’s up to us to make sure the minimum memory is properly charged for.
